  1. Que es un sistema de control de versiones?

    Son las herramientas de software que ayudan a los equipos de software a gestionar los cambios en el código fuente a lo largo del tiempo.

  2. Cual es la diferencia entre git y github?

    Git: es una herramienta de codigo abierto que los desarrolladores instalan localmente para la gestion del codigo.

    Github: es una plataforma basada en la nube, servicio en linea al que los edsarrolladores que usan GIT pueden conectarse y cargar o descargar kis recursos.

  3. Cómo creo una rama en git?

    Para crear la rama el git usa el comando "git checkout -b carpetarama/nuevarama"

  4. Cómo clono un repositorio?

    Para clonar el repositorio usa el comando "git clone" ejempleo: git@ip_del_server:nombre_repo.git
